CLI tool to scaffold a production-ready Node.js backend in seconds.

## Features
- Express server setup
- JWT authentication
- Cookie-based auth
- MongoDB connection
- OTP verification
- Error handlers
- Supports JavaScript & TypeScript

## Install
```bash
npm install -g create-node-app-latest
```
## Usage

#1 Help command
```bash
node-app -help
```

#2 Javascript Template
```bash
node-app myapp
```

#3 Typescript Template
```bash
node-app myapp --ts
```

#4 Template with Installed Dependency 
```bash
node-app myapp --install
```
```bash
node-app myapp --ts --install
```
